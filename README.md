📄 Recipe Page Project
This is a simple web page created as part of my Front-end Development training. The goal of this project was to convert a recipe image into a structured layout using only basic HTML and CSS.

🛠️ Technologies Used
HTML5: For the content and structure.
CSS3: For styling, colors, and layout (Flexbox).

Features
Clean and readable recipe layout.

Side-by-side design using CSS Flexbox.
